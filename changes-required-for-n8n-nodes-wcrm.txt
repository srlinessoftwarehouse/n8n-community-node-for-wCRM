[HIGH] httpRequest() used instead of httpRequestWithAuthentication() — blocks scanner

File: nodes/Wcrm/GenericFunctions.ts:33
this.helpers.httpRequest() is used alongside manual credential extraction via this.getCredentials(). n8n requires using this.helpers.httpRequestWithAuthentication('wcrmApi', options) instead, which delegates authentication to n8n's secure layer and enables features like audit logging and token refresh. This is enforced by the automated scanner and is a hard blocker. See HTTP helpers.
[HIGH] Webhook path hardcoded — webhookPath UI parameter has no effect

File: nodes/Wcrm/WcrmTrigger.node.ts:58-63

Either remove the webhookPath parameter and document the static path, or make the path dynamic by changing the webhooks[] entry to path: '={{$parameter["webhookPath"]}}'.
